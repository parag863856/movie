Movie Recommended System

Thanks For Visting to Our Website and for showing kind interest to us.

This Movie Recommended Application project is purely based on Machine Learning algorithms

In this application different kind of python library such as numpy, pandas, Sickit learn etc are used to make a machine learning model

For making this model, first we collect the training dataset from Kagle website and different types of datapreprocessings are done for making it higher accuracy

Here this application recommends similar kind of four movies based on user's choice

For this recommendation from the dataset of the model, vectorization and similarity methods are used

After making the recommendation ML model, We use Web-Dev technology to give this project an Userinterfac(UI)

In Frontend part we use HTML, CSS and JavaScripts Technology. On the otherhand Flask python library is used for Backend part

Through the Flask library we maintained the serverside and renders the recommendation to the Frontend Interface
